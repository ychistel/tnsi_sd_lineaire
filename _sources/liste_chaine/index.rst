
Les listes chainées
===================

Les listes chainées sont aussi une structure de donnée linéaire. Les valeurs qu'elle contient sont liées entre elles. Pour accéder à l'une d'elle, il faut passer par les autres en commençant par la première valeur de la liste. Chaque valeur est comme un maillon d'une chaine.
