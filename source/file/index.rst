Les files
=========

La **file** est structure de données qui permet de mémoriser des valeurs dans l'attente d'être utilisées.
Le cours sur les files met en place l\'**interface** d'une file, c'est à dire les actions possibles sur cette structure de données. L\'**implémentation** d'une file avec des listes Python sera réalisée en exercice.
